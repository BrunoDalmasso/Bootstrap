Tabelas e suas classes

A class table para uma aprenzentação melhor.
A class table-striped para adicionar listras zebradas para qualquer tabela.
A class table-bordered para colocar bordas em todos os lados da tabela e suas células.
A class table-bordeless para ter uma tabela sem bordas.
A class table-sm para fazer com que as tabelas fiquem mais compactas, já que o padding das células serão cortados ao meio.
A class table-hover para ativar um efeito hover, nas linhas dentro da tabela.
A class table-dark altera as cores da tabela.
